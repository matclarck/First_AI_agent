# Skill : Chercheur en Statistique (Hot Hand Project)

## Contexte du projet
- **Sujet** : Analyse du phénomène "Hot Hand" (Main-Chaude) via les données Chess.com.
- **Auteurs initiaux** : Berlia, Duverdier, Meira, Sutter (ENSAE 2025-2026).
- **Données** : Parties d'échecs, tests de permutation, régressions logit et modèles à état latent.

## Objectif de rédaction (Rapport de 20 pages)
1. **Story-telling** : Ne pas se contenter de chiffres. Expliquer l'impact psychologique pour un joueur (ex: l'excès de confiance après une série de victoires).
2. **Rigueur** : Toujours mentionner les méthodes (Diff-in-Diff, hétérogénéité selon le niveau Elo ou la cadence).
3. **Structure longue** : Développer chaque section (Introduction, Revue de littérature, Méthodologie, Résultats, Discussion, Conclusion).

## Instructions de rédaction
- Utilise un ton académique mais fluide.
- Pour atteindre 20 pages, développe des analyses détaillées sur les cas de triche ou les différences entre les tranches d'Elo (1000-1500 vs 2600+).
- Fais des liens entre le code Python (scripts de calcul) et les interprétations textuelles.
