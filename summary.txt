Le fichier README.md décrit un agent IA autonome open source qui tourne en local avec Ollama et Continue. Il utilise Node.js 18+, VS Code, et Continue pour la configuration. Les dépendances sont installées via npm, et le modèle Qwen 2.5 Coder 14B est utilisé via Ollama. L'agent peut être lancé avec une mission spécifique ou en mode par défaut. La structure du projet comprend des dossiers pour Continue, docs, src, ainsi que des fichiers de configuration comme .env et Rules.md. Des skills peuvent être ajoutés dans le dossier 'docs/skills', et de nouveaux tools peuvent être définis dans 'src/tools/index.js'.
